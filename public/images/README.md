# Portfolio Images

Replace placeholder images by adding your photos here:

```
public/images/
  hero-main.jpg
  hero-secondary.jpg
  portfolio/
    bridal-01.jpg
    soft-glam-01.jpg
    ...
```

Then update `data/portfolio.ts` to use local paths:

```ts
src: "/images/portfolio/bridal-01.jpg"
```

Recommended: WebP or JPEG, max 2000px wide, compressed for web.
